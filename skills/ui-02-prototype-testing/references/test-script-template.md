# Usability Test Script Template

**Feature:** … **Prototype link:** … **Duration:** 45 min

## Intro (5 min)
"We're testing the design, not you. Nothing you do is wrong. Please think out loud as you go. We may stay silent; that's normal." Consent for recording; anonymized notes only.

## Warm-up (5 min)
Context questions about their current behavior in this domain.

## Tasks (30 min)
Per task: scenario (goal language, no UI words) → observe → neutral probes only:
- "What are you looking at?"
- "What do you expect will happen?"
- "What would you do next?"
Never: "Did you see the button?" or any hint.

If stuck 2+ minutes: mark task failed, ask what they expected, then assist and continue.

## Wrap-up (5 min)
Single-ease question per task (1-7), overall impressions, "if you had a magic wand…"
